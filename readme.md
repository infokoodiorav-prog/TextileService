# Textile Service Website

The Textile Service website is a multi-page site demonstrating interactive elements and a multilingual user interface.

Multilingual support is implemented (Estonian, Russian, English, Finnish). Some translations are incomplete as the project is intended for demonstration purposes.

### Interactive and Technical Features

- Multilingual support: Estonian, Russian, English, Finnish
- Clickable images and popups on the Solutions page
- Clickbot that helps users select a service/product and automatically fills the form
- Slider, forms, and dynamic elements

### Technologies Used

- HTML, CSS, JavaScript
- JSON files for translations
